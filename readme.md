## 1. 저장 및 종료

```vim
:w             저장
:w file.txt    file.txt 파일로 저장
:w >> file.txt file.txt 파일에 덧붙여 저장
:q             종료
:q!            강제 종료
ZZ             저장 후 종료
:wq!           강제 저장 후 종료
:e file.txt    file.txt 파일 불러오기
:e             현재 파일 다시 불러오기
:e#            이전 파일 불러오기

a   커서 다음칸부터 입력
A   커서 행의 끝에서 입력
i   커서 위치에서 입력
I   커서 행의 처음에서 입력
o   다음 행에 입력
O   이전 행에 입력
s   커서 문자를 지우고 입력
cc  커서 행 전체 지우고 입력


h l          ← →
j k          ↓ ↑
w / W        다음 단어 첫 글자
b / B        이전 단어 첫 글자
e / E        단어의 마지막 글자
^            행의 첫 글자
$            행의 마지막 글자
+ / -        다음/이전 행 첫 글자
( / )        이전/다음 문장의 첫 글자
{ / }        이전/다음 문단으로 이동
H / M / L    화면 위/중앙/아래로 이동
z<CR> / z. / z-  현재 행을 화면의 위/중앙/아래로
[n]H / [n]L  위/아래에서 n번째 줄로 이동
Ctrl+u / d   반 화면 위/아래 스크롤
Ctrl+b / f   한 화면 위/아래 스크롤
gg / 1G      문서 맨 처음
G            문서 맨 끝
[n]G / :[n]  n번째 줄로 이동

x / dl       커서 위치 글자 삭제
X / dh       커서 앞 글자 삭제
dw           단어 삭제
diw          현재 단어 삭제
dd           한 줄 삭제
[n]dd        아래 n줄 삭제
dj / [n]dj   현재+아래 줄(n+1줄) 삭제
dk / [n]dk   현재+위 줄(n+1줄) 삭제
D / d$       커서~줄 끝 삭제
d0 / d^      커서~줄 처음 삭제


yy / Y       한 줄 복사
[n]yy        이후 n줄 복사
p            붙여넣기 (아래)
P            붙여넣기 (위)
[n]p         n번 붙여넣기

v            문자 단위 선택
V            줄 단위 선택
Ctrl+v       비주얼 블록 (윈도우: Ctrl+q)

선택 후 명령:
y            복사
d            삭제
r            치환
U / u        대문자 / 소문자
~            대소문자 전환
J            행 합치기
:            선택 영역 ex명령
< / >        들여쓰기 감소/증가
